# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - Production Release
### Added
- Progressive Web App (PWA) support with Offline mode.
- Analytics & Reporting module (Excel, PDF export).
- Firebase Security Rules and robust Authentication.
- Complete CI/CD Pipeline via GitHub Actions.
- Lazy Loading for all routes to optimize performance.
- Terms of Service and Privacy Policy pages.
- Setup Vitest for automated testing.
